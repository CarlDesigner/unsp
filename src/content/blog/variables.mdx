## Guía Básica de Variables en Programación

![](../../assets/images/variablesp.webp)

### ¿Qué es una variable?

En programación, una **variable** es como una caja que usamos para guardar información. Esa información puede cambiar a lo largo del tiempo, y podemos acceder a ella cuando la necesitemos. Es una de las bases más importantes que debes entender para aprender a programar, ya que las variables te permiten **almacenar, modificar y utilizar datos** en tus programas.

### ¿Por qué son importantes las variables?

Las variables son esenciales porque nos permiten **manipular datos** de manera dinámica. Sin ellas, sería muy difícil o imposible realizar tareas comunes en programación, como:

- Almacenar el nombre de un usuario.
- Guardar resultados de cálculos.
- Seguir la puntuación de un juego.

Con variables, puedes hacer que tu programa sea más flexible y adaptarse a diferentes escenarios.

### Sintaxis de las Variables en JavaScript

JavaScript utiliza tres palabras clave principales para declarar variables:

1. `var` (antiguo, no recomendado).
2. `let` (moderno, recomendado).
3. `const` (para variables que no cambian).

A continuación, veremos cómo se usa cada uno.

### Declaración de una Variable

Para declarar una variable en JavaScript, usamos la palabra clave `let` o `const`, seguida de un nombre que damos a la variable.

```javascript
let nombre // Declara una variable llamada 'nombre'
nombre = 'Juan' // Le asigna el valor 'Juan'
```

O también podemos declarar y asignar el valor en una sola línea:

```javascript
let nombre = 'Juan' // Declara y asigna el valor 'Juan'
```

### Tipos de Datos

Las variables pueden almacenar diferentes **tipos de datos**. Aquí algunos ejemplos:

1. **Números (Number):**

```javascript
let edad = 25
```

2. Cadenas de texto (String):

```javascript
let saludo = 'Hola, ¿cómo estás?'
```

3. Valores booleanos (Boolean):

```javascript
let esMayorDeEdad = true
```

### Ejemplo Práctico

Imagina que estás creando un programa sencillo para saludar a un usuario dependiendo de su nombre y edad.

```javascript
let nombre = 'Carlos' // Declaramos la variable 'nombre'
let edad = 28 // Declaramos la variable 'edad'

if (edad >= 18) {
	console.log(`Hola ${nombre}, eres mayor de edad.`)
} else {
	console.log(`Hola ${nombre}, aún no eres mayor de edad.`)
}
```

En este ejemplo:

- `nombre` guarda el nombre del usuario.
- `edad` guarda la edad del usuario.
- Dependiendo de si el valor de `edad` es mayor o igual a 18, mostramos un mensaje diferente.

### Reglas para Nombrar Variables

1. **No pueden comenzar con un número**. _Correcto:_ `edadUsuario`; _Incorrecto:_ `1usuario`.
2. **No pueden tener espacios**. _Usa camelCase:_ `nombreUsuario`.
3. **Son sensibles a mayúsculas y minúsculas**: `edad` y `Edad` son variables diferentes.

## Variables Constantes

En programación, cuando sabes que el valor de una variable **no debe cambiar** a lo largo de la ejecución del programa, lo recomendable es usar `const` para declararla. Este tipo de variables se denominan **constantes**. Un buen ejemplo es el valor de **PI** en matemáticas, que siempre es el mismo:


```javascript
const PI = 3.1416
```
Una vez que defines una variable con `const`, no puedes reasignarle un nuevo valor. Si intentas hacerlo, el programa generará un error. Las constantes son especialmente útiles cuando necesitas asegurarte de que ciertos valores clave no se modificarán accidentalmente durante la ejecución del código, lo que ayuda a mantener la estabilidad y seguridad en el programa.

Si intentas cambiar el valor de una variable declarada con `const`, obtendrás un error.

### Conclusión

Las variables son fundamentales en cualquier lenguaje de programación. Son como pequeños contenedores donde guardamos información que el programa necesita. Aprender a usarlas correctamente te permitirá escribir programas más dinámicos y útiles.

#### Ejercicio Práctico 1: Área de un rectángulo

*Objetivo:* Usar variables para calcular el área de un rectángulo.

*Instrucciones:*
1. Declara dos variables, `largo` y `ancho`, y asígnales valores numéricos.
2. Declara una tercera variable, `area`, que contenga el resultado de multiplicar `largo` por `ancho`.
3. Muestra el área en la consola.

<details className="p-4 rounded-md bg-gray-100 dark:bg-gray-800">
<summary className="cursor-pointer text-blue-600 dark:text-blue-400 hover:underline">Ver resultado</summary>
```javascript

let ancho = 4;

let area = largo * ancho;

console.log(area);  // Resultado: 32

```
</details>
